# Carpet & Tile Ops — PWA Business App

A fully offline-capable Progressive Web App (PWA) for managing your carpet & tile cleaning business. Works on **Android, iOS, Windows, and macOS** — no subscription, no server needed.

## 🚀 Free Hosting & Installation Guide

### Option 1: GitHub Pages (Recommended — Completely Free)

1. Create a free account at [github.com](https://github.com)
2. Create a new repository named `carpet-tile-ops`
3. Upload all files from this folder to the repository
4. Go to **Settings → Pages → Source: main branch / root folder**
5. Your app will be live at: `https://yourusername.github.io/carpet-tile-ops`

### Option 2: Netlify (Also Free)

1. Go to [netlify.com](https://netlify.com) → Sign up free
2. Drag and drop this entire folder onto the Netlify dashboard
3. Your app gets a URL like `https://random-name.netlify.app`

---

## 📲 Install on Any Device (after hosting)

| Platform | Steps |
|----------|-------|
| **Android** | Chrome → tap ⋮ → **Install App** / **Add to Home Screen** |
| **iPhone/iPad** | Safari → tap Share 📤 → **Add to Home Screen** |
| **Windows PC** | Chrome/Edge → click ⊕ icon in address bar → **Install** |
| **Mac** | Chrome → click ⊕ icon → **Install** |

Once installed, the app works **100% offline** — all data stored locally on the device.

## Features

- **Dashboard**: Net-to-Keep, monthly navigation (‹ ›), service breakdown with splits
- **Ledger**: Income/expense tracking, auto business/personal categorization, receipt photos
- **Tracker**: Job timer with GPS, equipment maintenance & sinking fund
- **Tax**: CRA T2125 summary, ITC credits, year-over-year comparison
- **Settings**: Per-service split configuration (38% carpet, 45% tile, $25/hr hourly)

## Local Development

For local use without hosting, just open `index.html` in a browser. The PWA install feature requires HTTPS (GitHub Pages or Netlify provide this for free).
